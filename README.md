# parcial1-LuisFelipe-Saldarriaga

Aplicación web para gestionar compras y productos de una tienda de tecnología.
Tecnologías usadas

HTML5
CSS3
JavaScript
Bootstrap 5
Git / GitHub

Funcionalidades

Catálogo de productos con cards por categoría
Botón para agregar productos al carrito con modal informativo
Formulario de contacto con validación RegEx
Modal con la información diligenciada al enviar el formulario
Modo oscuro / modo claro
